## :checkered_flag: Starting

```bash
# Clone this project
$ git clone https://github.com/mrfelipelima/labgefabox-backend

# Access
$ cd labgefabox-backend

# Install dependencies
$ yarn

# Run the project
$ yarn start

# The server will initialize in the <http://localhost:3000>
```
